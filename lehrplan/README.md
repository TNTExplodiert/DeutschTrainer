# 📚 Lehrplan-Referenz (Bayern, LehrplanPLUS)

Strukturierte Markdown-Aufbereitung der **frei zugänglichen, amtlichen** Lehrplan-Inhalte
des bayerischen *LehrplanPLUS* – als Grundlage für eigene, lehrplankonforme Übungen
in diesem Projekt.

Quelle: [lehrplanplus.bayern.de](https://www.lehrplanplus.bayern.de/) (Staatsinstitut für
Schulqualität und Bildungsforschung, ISB / Bayerisches Staatsministerium für Unterricht und Kultus).

## ⚖️ Rechtlicher Rahmen (wichtig)

- **Amtliche Lehrplan-Texte** (Kompetenzbeschreibungen, Inhalte, Fach-/Jahrgangsstufenprofile)
  werden hier aufbereitet, weil sie als **amtliches Werk** frei zugänglich und nutzbar sind.
- **Alles andere wird nur verlinkt, nicht kopiert:** keine Schulbuch-/Verlagsinhalte,
  keine kommerziellen Arbeitsblätter, keine geschützten Vokabel-/Textsammlungen.
- Frei nutzbare Materialien (CC-Lizenz, Public Domain, behördlich) werden mit
  **Quelle + Lizenz** als Link geführt – zur Orientierung beim Erstellen *eigener* Inhalte.

## 🗂️ Struktur

```
lehrplan/<schulart>/<jahrgangsstufe>/<fach>/
  lehrplan.md            – amtliche Kompetenzen & Inhalte (aus LehrplanPLUS)
  wortschatz-themen.md   – Themenfelder + freie Wortquellen (nur Links)  [Sprachen]
  freie-materialien.md   – kuratierte, frei lizenzierte Materialien (nur Links)
```
