# assets/

## Optionaler Nine-Circles-Song (Bonus-Level „NINE")

Wenn du den **echten** Song im NINE-Level hören möchtest, lege ihn hier ab:

```
game/assets/nine.mp3
```

- Das Spiel erkennt die Datei automatisch und spielt sie im NINE-Level
  **von Anfang an (0:00)** ab; bei einem Crash startet sie wieder am Anfang.
- Ist **keine** Datei vorhanden, läuft ein **eigener, lizenzfreier** Track
  im Nine-Circles-Stil als Fallback (Standard).

### Wichtig – Lizenz/Urheberrecht
Der gewünschte Track (Newgrounds-Audio, z. B. ID **574484**) ist
urheberrechtlich geschützt. Lade/verwende ihn **nur**, wenn die Nutzungs-
bedingungen des Künstlers das erlauben. Diese Datei ist hier **bewusst nicht**
enthalten – du müsstest sie selbst (legal) als `nine.mp3` hinzufügen.
